BurnHall2.0 框架
========================
# 简介
BurnHall2 是基于laravel7 和 Ant Design 开发的一套前后端分离的后台基础功能集成框架

#前端
```
# 初始化项目
yarn install

# 开发
yarn serve

# 编译
yarn build
```

#功能  
-[ ] 注册登陆 用户管理 权限管理  
-[x] ~~三层架构~~ + redis表级缓存   
-[x] sql日志 sql慢查询日志 api请求日志
-[x] 自动匹配 `Restful API` 路由
-[ ] 定时任务  
-[ ] 消息队列  
-[ ] 微信封装  
-[ ] 日志封装
